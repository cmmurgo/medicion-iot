# Proyecto de Medición IoT

Este proyecto consta de dos partes:

## 1. Sensores (Frontend)
Aplicación React para la visualización y gestión de tanques en un canvas.
- Ubicación: `/sensores`
- Iniciar: `npm run dev`

## 2. WS-Sensores (Backend)
Web Service desarrollado en Yii2 con PostgreSQL.
- Ubicación: `/ws-sensores`
- Iniciar: `php yii serve --port=5000`

### Requisitos
- PHP 7.4+
- Composer
- PostgreSQL
- Node.js & NPM
